关于virtualbox安装完成之后出现的Connot access the kernel driver错误的解决办法

想必很多童鞋更新或者重装vbox之后安装虚拟机的时候就会出现标题上出现的问题，那么如何解决呢？



看看方法吧


首先是，进入到C:\Program Files\Oracle\VirtualBox\drivers\vboxdrv文件夹之中，就是你的vbox的目录中的drivers\vboxdrv目录，之后找到vboxdrv.inf文件，双击安装即可解决这个问题。
